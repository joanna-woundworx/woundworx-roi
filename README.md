## Deploy to Vercel

### Option A — CLI (fastest)
```bash
npm install -g vercel
npm install
vercel --prod
```
Project name: `woundworx-roi` → URL: `https://woundworx-roi.vercel.app`

### Option B — No code
1. vercel.com → New Project → Upload this folder
2. Framework auto-detected as Create React App
3. Click Deploy
